## Installation Require Dependencies

**Install `openpyxl`**

```sh
pip install openpyxl
```

**Install `pywin32`**

```sh
pip install pywin32
```

## Installation `.exe` file

**Install `pyinstaller`**

```sh
pip install pyinstaller
```

**Creation `.exe` file**

```sh
pyinstaller --onefile --noconsole --icon=your_icon.ico your_script.py
```
